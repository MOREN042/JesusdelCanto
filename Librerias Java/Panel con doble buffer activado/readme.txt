Esta clase hereda de JPanel y crea un Panel con el doble búfer activado para el correcto funcionamiento de gráficos en movimiento.

Constructor:

ventanaDobleBufer ([int] tamaño horizontal del Jpanel, [int] tamaño vertical del Jpanel, [int] posición horizontal, [int] posición vertical).

Métodos:

cambiarColor ([Color] color que queremos que tenga de fondo).

El resto de métodos los hereda directamente de la clase JPanel.
